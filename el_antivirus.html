El antivirus

Windows incluye un antivirus llamado Microsoft Defender. Este se encarga de examinar continuamente en busca de malware (software malintencionado), virus y amenazas de seguridad. 

Si tenemos instalada y activada otra aplicación antivirus, el Antivirus de Microsoft Defender se desactivará automáticamente. Si desinstalamos la otra aplicación, Antivirus de Microsoft Defender se volverá a activar automáticamente.

Para personalizar el antivirus podemos hacer doble clic en el icono de Seguridad de Windows que se encuentra en la barra de tareas , o bien, seleccionar Inicio  > Configuración  > Actualización y seguridad  > Seguridad de Windows.

Podemos realizar diferentes tipos de análisis sobre nuestro sistema: análisis rápido, completo o selectivo.

Es posible analizar un archivo o carpeta desde el explorador de archivos. Basta con seleccionar el elemento (o varios) y hacer clic con el botón derecho sobre él. Aparecerá un menú contextual con la opción de realizar un análisis del elemento. Al terminar el análisis, el antivirus nos mostrará el resultado del mismo. En el caso de que el archivo esté infectado, el antivirus nos ofrecerá diferentes acciones para mantener a raya el virus.
