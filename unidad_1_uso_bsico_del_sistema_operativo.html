Unidad 1: Uso básico del sistema operativo

Cuando encendemos el ordenador, lo primero que tiene lugar es una prueba de diagnóstico que examina la memoria del ordenador, el procesador y otros dispositivos importantes.  Esta prueba se denomina con las siglas P.O.S.T. (Power-On-Self-Test). 

Quien se encarga de realizar el POST es la BIOS. Si esta detecta un problema grave, avisa con una serie de pitidos y no permite que se cargue el sistema operativo. Algunas placas base modernas incluyen un display que muestra el código de error, que luego podemos consultar en el manual de la placa base para determinar cuál es el problema.

En otro caso, si todo funciona correctamente, la BIOS pasa el control al sistema operativo.

¿Qué es la BIOS?

La BIOS (Basic Input Output System) es un pequeño chip de memoria programable que se encuentra en la placa base. Posee un software que permite configurar varias opciones del hardware. 

Actualmente podemos encontrar dos tipos de BIOS:

  • BIOS Legacy: es más antigua y en su menú solo podemos movernos mediante teclado.
  • BIOS UEFI: es la que traen las placas base modernas, suelen basarse en una interfaz de usuario más amigable, con más información y que se puede manejar con teclado y ratón.  

¿Para qué sirve entrar en la BIOS?
En el caso de que queramos configurar algún parámetro del hardware, como el orden de arranque de nuestro equipo o los modos de ahorro de energía del procesador, ... Ahora bien, no es recomendable hacer cambios si no estamos seguros, ya que el equipo puede dejar de funcionar correctamente tras los cambios.

¿Cómo se accede a la configuración de la BIOS del PC?

Nada más pulsar el botón de encendido, hay que pulsar una tecla: ESC, F1, F2, Supr , F10..., depende del fabricante de la BIOS. Se pulsa hasta que aparezca la pantalla de configuración. 

Como a veces no da tiempo a pulsar la tecla, otra forma más fácil es desde el sistema operativo. En Windows: menú Inicio-Configuración-Actualización y seguridad-Recuperación-Reiniciar ahora-Solucionar problemas-Configuración de firmware UEFI.

    ¿Qué es un sistema operativo?

    Es el software más importante que se ejecuta en un ordenador. Sin él, el PC sería una máquina inútil para nosotros, ya que nos permite usarlo y darle órdenes para que haga lo que necesitamos. Cuando compramos un ordenador, el sistema operativo suele venir ya instalado.

    ¿Qué hace el sistema operativo?

    Se encarga de administrar los recursos del ordenador, es decir, el software y hardware. Es el que soporta y maneja todos los programas y partes del ordenador.

    Los sistemas operativos modernos presentan una interfaz gráfica de usuario (GUI, Graphical User Interface), para que el usuario pueda interactuar con el ordenador, por ejemplo, utilizar el ratón para hacer clic sobre los iconos y botones, o interaccionar con cualquier otro elemento para ejecutar acciones o tareas... 

    La interfaz gráfica de usuario de cada sistema operativo tiene un aspecto diferente, por lo que al cambiar de un sistema a otro puede resultarnos extraño al principio, pero todos tienen funciones muy similares.

    Nosotros trabajaremos con el sistema operativo Windows, no obstante, es el más usado en los ordenadores personales en el ámbito doméstico.
